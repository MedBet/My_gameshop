# 🐻 BearPass — Онлайн-магазин подписок и цифровых товаров

**BearPass** — это компактный веб-сайт электронной коммерции, реализованный на чистом PHP, MySQL, HTML и CSS. Проект позволяет продавать цифровые подписки (игры, музыка, спорт, кино и т.д.) с полным циклом оформления заказа, включая корзину, управление заказами и административную панель.

Этот проект был создан как учебный/демонстрационный пример работы интернет-магазина без использования фреймворков.

---

## 🚀 Основные возможности

### Для покупателей

*   🔐 **Регистрация и аутентификация**: создание учётной записи и вход в систему.
*   🏠 **Каталог товаров**: просмотр подписок с разбивкой по категориям.
*   🔍 **Фильтрация и поиск**: поиск по названию или компании, сортировка по цене и наименованию.
*   🛒 **Корзина**: добавление/удаление товаров, объединение в один заказ.
*   📦 **Оформление заказа**: автоматическая генерация текстового файла с деталями заказа.
*   📋 **История заказов**: просмотр всех оформленных заказов и их содержимого.

### Для администратора

*   ➕ **Управление подписками**: добавление, удаление товаров с загрузкой изображений.
*   📊 **Просмотр заказов**: доступ ко всем заказам пользователей (файл `admin_subscribes.php`).
