🐻 BearPass — Онлайн-магазин подписок и цифровых товаров

BearPass — это компактный веб-сайт электронной коммерции, реализованный на чистом PHP, MySQL, HTML и CSS. Проект позволяет продавать цифровые подписки (игры, музыка, спорт, кино и т.д.) с полным циклом оформления заказа, включая корзину, управление заказами и административную панель.

Этот проект был создан как учебный/демонстрационный пример работы интернет-магазина без использования фреймворков.

🚀 Основные возможности

Для покупателей

🔐 Регистрация и аутентификация: создание учётной записи и вход в систему.

🏠 Каталог товаров: просмотр подписок с разбивкой по категориям.

🔍 Фильтрация и поиск: поиск по названию или компании, сортировка по цене и наименованию.

🛒 Корзина: добавление/удаление товаров, объединение в один заказ.

📦 Оформление заказа: автоматическая генерация текстового файла с деталями заказа.

📋 История заказов: просмотр всех оформленных заказов и их содержимого.

Для администратора

➕ Управление подписками: добавление, удаление товаров с загрузкой изображений.

📊 Просмотр заказов: доступ ко всем заказам пользователей (файл admin_subscribes.php).

🗄️ Структура базы данных

Схема базы данных описана в файле Db.xlsx. На основе анализа кода выделяются следующие таблицы:

Таблица

Назначение

user

Учётные записи пользователей (login, password, mail, is_admin)

type

Типы подписок (музыка, игры, спорт и т.д.)

company

Компании-поставщики услуг

subscribes

Сами подписки (name, time, price, type_id, company_id)

photo

Загруженные изображения (name, link)

photo_catalog

Связь many-to-many между подписками и фотографиями

order

Заказы пользователей (data, check, user_id)

cart

Состав корзины (order_id, subscribes_id)

⚙️ Установка и запуск

1. Требования

Веб-сервер с поддержкой PHP 7.0+ (например, XAMPP, OpenServer или Apache).

MySQL 5.7+ (или MariaDB).

Расширение PHP mysqli.

Права на запись в папку images/.

2. Настройка базы данных

Создайте базу данных (например, bearpass_shop).

Создайте таблицы согласно схеме в Db.xlsx.

3. Конфигурация

Отредактируйте файл config.php:

<?php
$host = 'localhost';
$user = 'ваш_пользователь';
$pass = 'ваш_пароль';
$db   = 'bearpass_shop';

$con = mysqli_connect($host, $user, $pass, $db);
if (!$con) {
    die('Ошибка подключения: ' . mysqli_connect_error());
}
mysqli_set_charset($con, 'utf8');
?>


4. Размещение и права

Скопируйте файлы в htdocs или public_html. Установите права на папку с картинками:

chmod 755 images/


5. Права администратора

Зарегистрируйте аккаунт и выполните SQL-запрос, чтобы стать админом:

UPDATE user SET is_admin = 1 WHERE id = 1;


⚠️ Важные замечания по безопасности

Данный проект является учебным и не предназначен для коммерческого использования без доработок.

Пароли: В текущей версии пароли могут храниться ненадёжно. Используйте password_hash() и password_verify().

SQL-инъекции: Рекомендуется переписать запросы на PDO или использовать mysqli_prepare.

XSS/CSRF: Требуется внедрение фильтрации вывода и защитных токенов для форм.
